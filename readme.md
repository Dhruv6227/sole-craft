# 👟 Sole Craft

Sole Craft is a modern footwear showcase web application that allows users to explore stylish shoe collections with a clean and responsive interface.  
It also includes a simple admin panel to manage content.

🔗 Live Demo: https://sole-craft-nu.vercel.app

---

## 🚀 Features

- 🛍 Modern product showcase layout
- 📱 Fully responsive design
- ⚡ Fast loading static website
- 🔐 Admin panel (admin.html)
- 🎨 Clean UI using CSS
- 📂 Organized file structure
- 🌐 Deployed on Vercel

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Vercel (Deployment)

---

## 📁 Project Structure


sole-craft

│── admin.html # Admin panel
│── styles.css # Styling file

│── admin.js # Admin panel logic
│── images/ # Product images

3. Run project

Open index.html in browser

## 🌍 Deployment

This project is deployed using **Vercel**.

If you want to deploy:
1. Push code to GitHub
2. Go to vercel.com
3. Import repository
4. Click deploy
